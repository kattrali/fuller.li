Projects

I'm a part of the core team in many open source projects such as Nest, and Pelican, along with many others.

Swift on the Server

My goal is building a flexible, type-safe, and cross-platform future for Swift in servers and backend frameworks.

  • Nest

    Nest is an interface for interaction between web servers and web applications or frameworks supporting swift, to enable the use of any application with any server. It has a number of implementing projects including web servers, frameworks, and example middlewares.

  • Stencil

    Stencil is a simple and powerful template language for Swift which provides a syntax similar to Django and Mustache.

  • Heroku Buildpack for Swift

    A buildpack for easy deployment of Swift code to Heroku

Better Swift Testing

Testing in Swift should be easy and ergonomic for mobile and command-line apps.

  • Spectre

    A cross-platform behavior-driven development (BDD) framework and test runner for Swift projects and playgrounds.

  • Mockingjay

    An elegant library for stubbing HTTP requests in Swift

  • Ploughman

    A Swift implementation of Cucumber, a test runner for the Gherkin language.

Specification-driven Software Design

Formal specifications for software design and development leads to better design and usability for the components of software systems.

  • apiblueprint.vim

    API Blueprint is a high-level API description language for web APIs, built on Markdown. This Vim plugin adds support for API Blueprint language syntax highlighting and linting.

  • Changelog

    A formal specification for semantic changelog files and entries, for ease of reading and automation via tools.
